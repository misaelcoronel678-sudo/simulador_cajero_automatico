# 🏦 Simulador de Cajero Automático

Este es un proyecto desarrollado en **Python** para simular las operaciones básicas de un cajero automático paraguayo.

### 🚀 Funcionalidades
* **Validación de saldo:** Verifica si el usuario tiene fondos suficientes.
* **Estado del cajero:** Simula si la máquina tiene efectivo.
* **Seguridad:** Bloquea retiros de montos negativos.
* **Formato profesional:** Resultados en Guaraníes (Gs) con separador de miles.

### 🛠️ Tecnologías
* Python 3
* Desarrollado usando Pydroid 3 desde un telefono Android.
